# COMICS APP

Proyecto del taller de ReactJs de Bits Americas S.A.S.

Dictado por Ludwring Liccien [@lliccien](https://twitter.com/lliccien)

## Instalación

`npm i` ó `npm install`

## Ejecución en modo de desarrollo

`npm start`

## Ejecución de EsLint

`npm run lint`

## Ejecución de Prettier

`npm run format`
